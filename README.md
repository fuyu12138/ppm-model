# Phenomenological Pragmatic Moral Model (PPM)
## 现象学实践主义伦理模型

PPM（Phenomenological Pragmatic Moral Model）是一种**后形而上学的实践理性伦理模型**。  
该模型在完全接受自然主义科学世界观（F₁）的前提下，将第一人称规范性经验（F₂）视为行动者不可消解的构成性条件，并以此为出发点构造其规范结构。

PPM 不诉诸任何形而上学实体、超验根据或价值本体论担保。  
其核心问题并非“何为善”，而是：  
**在不否定自然主义世界观的前提下，实践理性与规范性如何仍然可能。**

---

## 基本立场（概述）

- PPM 接受自然科学对世界的描述完备性，但否认第一人称规范性经验可以被还原为纯粹的第三人称事实。
- F₁（自然主义事实层面）与 F₂（第一人称规范性经验）被视为**不可通约但共存**的两个层面。
- 规范并非从价值实体或先验原则中推出，而是从**意义建构共同体得以持续运作的结构条件**中生成。

在此意义上，PPM 提供的是一种**实践理性得以成立的结构模型**，而非一套直接给出行动结论的规范清单。

---

## 元规范取向

PPM 以“**意义建构共同体的可持续性、韧性与包容性**”作为其元规范取向，并在此基础上约束其规范推导与实践判断。

该元规范并非价值宣言，而是对以下事实的回应：  
实践理性与规范性只能在意义得以被共同维持的社会—实践结构中持续存在。
